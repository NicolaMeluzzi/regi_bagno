DDL in SQL

Struttura e gestione dei database

Cos'è il DDL

Il DDL permette di creare e modificare database.

  • CREATE DATABASE
  • CREATE TABLE
  • ALTER TABLE
  • DROP TABLE

CREATE DATABASE

CREATE DATABASE Scuola;

DROP DATABASE

DROP DATABASE Scuola;

CREATE TABLE

CREATE TABLE Studenti (
 ID INT PRIMARY KEY,
 Nome VARCHAR(50) NOT NULL,
 Eta INT CHECK (Eta >= 6)
);

ALTER TABLE

ALTER TABLE Studenti
ADD Email VARCHAR(100);

DROP TABLE

DROP TABLE Studenti;

Modifiche avanzate

ALTER TABLE Studenti
DROP COLUMN Email;

ALTER TABLE Studenti
ADD CONSTRAINT chk_eta
CHECK (Eta >= 6 AND Eta <= 19);

Tipi di dato

  • INT, DECIMAL → numeri
  • VARCHAR → testo
  • DATE → date
  • BOOLEAN → vero/falso

Esempio tipi

CREATE TABLE Prodotti (
 ID INT PRIMARY KEY,
 Nome VARCHAR(50),
 Prezzo DECIMAL(6,2),
 DataCreazione DATE,
 Disponibile BOOLEAN
);

Vincoli SQL

  • NOT NULL
  • UNIQUE
  • PRIMARY KEY
  • CHECK

Esempio vincoli

CREATE TABLE Utenti (
 ID INT PRIMARY KEY,
 Nome VARCHAR(50) NOT NULL,
 Email VARCHAR(50) UNIQUE,
 Eta INT CHECK (Eta >= 18)
);
