Slide 9

Introduzione ai Vincoli

I vincoli (Constraints) sono regole applicate alle colonne di una tabella.

  • Servono a controllare l'integrità dei dati.
  • Impediscono l'inserimento di dati errati o inconsistenti.

Slide 10: NOT NULL & UNIQUE

NOT NULL: Obbliga la colonna ad avere sempre un valore. Non può essere lasciata vuota.

UNIQUE: Assicura che tutti i valori in una colonna siano distinti tra loro (niente duplicati).

Slide 11: PRIMARY KEY & CHECK

PRIMARY KEY

Identifica univocamente ogni riga. È un mix di NOT NULL e UNIQUE.

CHECK

Verifica che il valore soddisfi una condizione specifica (es. Età > 18).

Slide 12: Esempio Completo


CREATE TABLE Utenti (
    ID int PRIMARY KEY,
    Nome varchar(255) NOT NULL,
    Email varchar(255) UNIQUE,
    Eta int CHECK (Eta >= 18)
);
